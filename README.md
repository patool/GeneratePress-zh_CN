# 简介

GeneratePress主题汉化。

# 汉化指导

系统信息：

	WordPress版本：4.7.8
	主题generatepress版本：2.0.1
	汉化工具：Poedit 2.0.2

主要步骤：

  - 找一个主题，下载、解压；
  - 使用Poedit从主题创建翻译，逐条翻译后保存成`主题名称-zh_CN.po`和`主题名称-zh_CN.mo`；
  - 将`主题名称-zh_CN.po`和`主题名称-zh_CN.mo`拷贝到`wp-content/languages/themes`目录下；
  - 配置WordPress选择中文，刷新页面后即可看到汉化的界面；
